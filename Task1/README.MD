#Task 1. University Data Centre Process and Resource Management System

##Description
System administration menu:
1.Display current CPU and memory usage. 
2.List the top ten memory consuming processes. 
3.Terminate a selected process only after confirmation.
4.Imspect disk usage
4.Archive large log files
6.Bye(exit the program)
## How to run:
1. Make the script executable
command: chmod +x task1_process_resource_management.sh
2. Run the script
command:./task1_process_resource_management.sh
## Notes
- Logs are stored in `system_monitor_log.txt`.
- ArchiveLogs folder is created automatically for large logs.
