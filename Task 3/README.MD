##Description##

System for secure assignment submission using Bash + Python:

1.Submit assignments (.pdf or .docx only, <5MB).

2.Detect duplicate submissions using MD5 hashing.

3.List all submitted assignments.

4.Simulate login attempts with password (admin123).

5.Lock account after 3 failed attempts and detect suspicious activity within 60 seconds.

6.Logs all actions in submission_log.txt.

##HOW TO RUN####

1. make the script executeable
command:chmod +x secure_system.sh

2. run the main menu
command:./secure_system.sh
